# 将题库上传到老师Gitee仓库的步骤

## 前提条件
1. 你已经创建了自己的Gitee仓库
2. 你有老师Gitee仓库的地址（例如：https://gitee.com/老师用户名/课程仓库名.git）
3. 你的电脑已安装Git

## 操作步骤

### 步骤1：设置Git全局配置（如果尚未设置）
```bash
git config --global user.name "你的Gitee用户名"
git config --global user.email "你的邮箱"
```

### 步骤2：初始化本地Git仓库
```bash
# 在当前目录初始化Git仓库
git init

# 添加所有文件到暂存区
git add .

# 提交文件到本地仓库
git commit -m "初始提交：传播学概论题库"
```

### 步骤3：连接到老师的Gitee仓库
```bash
# 添加老师仓库作为远程仓库（将URL替换为老师提供的地址）
git remote add teacher https://gitee.com/老师用户名/课程仓库名.git
```

### 步骤4：推送文件到老师仓库
```bash
# 推送到老师仓库的main分支
git push teacher main
```

## 注意事项
1. 如果老师的仓库已有内容，可能需要先拉取：`git pull teacher main`
2. 如果推送失败，可能是权限问题，请联系老师获取推送权限
3. 推送成功后，通知老师查收

## 备选方案：如果没有推送权限
如果老师没有给你推送权限，可以使用以下方法：

1. 将题库文件打包为ZIP文件
2. 通过邮件或其他方式发送给老师
3. 或者使用Gitee的Pull Request功能（需要先Fork老师仓库）
